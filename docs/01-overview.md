# 项目概述

## 1. 项目简介

**Fortran LU Solver** 是一个高性能、稳定的 Fortran 库，专门用于执行 LU 矩阵分解并求解线性方程组。该项目采用部分选主元（Partial Pivoting）算法，确保数值稳定性，并包含完整的测试套件和实际应用示例。

### 1.1 项目定位

本项目旨在为科学计算和工程应用提供一个：

- **轻量级**：无外部依赖，纯 Fortran 实现
- **高性能**：针对稠密矩阵优化的分解算法
- **数值稳定**：采用部分选主元策略处理病态矩阵
- **易于集成**：模块化设计，便于嵌入现有项目

### 1.2 核心功能

| 功能模块  | 描述                  |
| ----- | ------------------- |
| LU 分解 | 带部分选主元的 LU 矩阵分解     |
| 线性求解器 | 完整的 Ax = b 求解接口     |
| 矩阵工具  | 矩阵生成、范数计算、残差检查等辅助功能 |
| 应用示例  | 电路分析等实际工程应用案例       |

## 3. 技术背景

### 3.1 LU 分解简介

LU 分解是将矩阵 A 分解为下三角矩阵 L 和上三角矩阵 U 的乘积：

```
A = L × U
```

其中：

- L 是单位下三角矩阵（对角线元素为 1）
- U 是上三角矩阵

### 3.2 部分选主元策略

为提高数值稳定性，本项目实现了部分选主元策略：

1. 在每步消元前，选择当前列中绝对值最大的元素作为主元
2. 必要时交换矩阵行，确保主元位置最优
3. 有效避免小主元导致的数值误差放大

### 3.3 求解流程

求解线性方程组 Ax = b 的完整流程：

```
┌─────────────────┐
│   输入矩阵 A    │
│   和向量 b      │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  LU 分解        │
│  (带部分选主元) │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  前向替换       │
│  Ly = Pb        │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  后向替换       │
│  Ux = y        │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   输出解向量 x  │
└─────────────────┘
```

## 6. 快速开始

### 6.1 环境要求

- Fortran 编译器（gfortran 推荐）
- make 构建工具

### 6.2 快速构建

```bash
# 克隆项目
git clone https://gitee.com/fluman2024/fortran_lu_solver.git
cd fortran_lu_solver

# 编译所有目标
make all

# 运行测试
make test

# 运行电路分析应用
./bin/circuit_solver
```

### 6.3 基本使用示例

```fortran
program demo
  use linear_solvers
  implicit none
  real, dimension(3,3) :: A
  real, dimension(3) :: b, x
  integer :: ierr

  ! 初始化矩阵和向量
  A = reshape([2.0, 1.0, 1.0, &
               4.0, 3.0, 3.0, &
               8.0, 7.0, 9.0], [3,3])
  b = [1.0, 2.0, 3.0]

  ! 求解方程组
  call solve_linear_system_with_pivot(A, b, x, ierr)

  if (ierr == 0) then
    print *, '解向量 x:', x
  else
    print *, '求解失败，错误代码:', ierr
  end if
end program demo
```
